# [M-Band-COO-X] SHX Watch — Reference Specifications & Iteration 3 Request

**Project:** M-Band-COO-X / Bom-Tier / Straps
**Supplier:** SHX Watch
**Status:** Open — awaiting supplier response
**Prepared by:** Sourcing + Engineering
**Reference strap:** Current strap (M-Band COO-CH) — MA12121-08 & MA12123-05
**Date:** 2026-05-13

## 1. Context

Following the 2nd round of samples received from SHX Watch and the supplier's readiness to invest in mould tooling to match our current straps, this document consolidates:

- The reference specifications for the strap.
- The deviations observed in the 2nd batch.
- The request for the next iteration before tooling investment.

The reference strap is currently in SHX Watch's possession, allowing for direct side-by-side comparison.

## 2. Reference Specifications

Target specifications, based on our current production strap:

| Aspect | Target |
| --- | --- |
| Width | 21.0 mm (+/- 0.5 mm) |
| Thickness | 2.0 mm (+0.5 / -0) |
| Velcro fabric — density | Equivalent to reference strap |
| Velcro fabric — surface finish | Smoother, higher loop density than 2nd batch |
| Velcro hooks — size | Smaller than 2nd batch, aligned with reference |
| Velcro hooks — spacing | Closer spacing, aligned with reference |
| Velcro hooks — finish | Matte (not glossy) |
| Seam at buckle end | Not visible on the outer side |
| Buckle | Dimensions matching reference strap |

## 3. Observed Deviations (2nd batch vs reference strap)

| # | Aspect | Reference | 2nd batch observation |
| --- | --- | --- | --- |
| 1 | Velcro fabric — density and thickness | Smoother surface, higher loop density, thinner fabric | Lower density, harsher feel, visibly thicker fabric |
| 2 | Velcro hooks — design, size and finish | Smaller hooks, closer spacing, matte finish | Larger hooks, wider spacing, glossier surface |
| 3 | Seam at buckle end | Not visible on the outer side | Visible on the outer side |
| 4 | Buckle dimensions | As per reference strap | Visibly different sizing |
| 5 | Overall dimensions | 21.0 mm width (+/- 0.5), 2.0 mm thickness (+0.5/-0) | Outside range on both axes |

All deviations are directly observable by placing the 2nd batch sample side by side with the reference strap.

## 4. Next Iteration Request

Before moving into mould tooling investment, SHX Watch is requested to provide one of the following:

**(a) Pre-tooling sample** — a sample that demonstrates the velcro fabric, hook design, and seam construction can match the reference using existing materials and processes. This validates the textile and assembly side before any mould investment.

**(b) Written technical proposal** — a document describing how SHX intends to achieve each deviation listed above (fabric sourcing, hook specification, seam construction, buckle tooling). This allows alignment on feasibility without committing resources prematurely.

Either path is acceptable. The objective is to ensure both parties have evidence that the reference specifications are achievable before tooling is engaged.

## 5. Information Requested from SHX

1. Expected timeline for the next iteration (pre-tooling sample or technical proposal).
2. MOQ and lead time at production volume, once specifications are confirmed.
3. Any constraints (material availability, fabric sourcing limits, tooling capability) that may prevent matching the reference specifications.

## 6. Iteration History

- **1st round assessment** — initial sample evaluation.
- **2nd round assessment** — basis for this document.

## 7. Rationale

This strap is going into production. Any deviation at sample stage tends to multiply at volume. Aligning on reference specifications before tooling is committed reduces risk for both parties and protects the investment SHX Watch is prepared to make in mould tooling.
